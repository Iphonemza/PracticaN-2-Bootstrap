Imagen de Windows 10

La utilizacion del software Sysprep presente en todas las instalaciones de Windows, permite editar configuraciones específicas del sistema operativo para llevarlas a un estado general y lo prepara para su captura, clonación y distribución en múltiples máquinas en una misma red. Esto garantiza que todo funciona correctamente y los equipos implementados mantienen una configuracion identica en un entorno de trabajo unificado.

Microsoft Blog
Samba Server

Samba es una implementación libre del protocolo de archivos compartidos de Microsoft Windows (antiguamente llamado SMB, renombrado recientemente a CIFS) para sistemas de tipo UNIX. De esta forma, es posible que computadoras con GNU/Linux, Mac OS X o Unix en general se vean como servidores o actúen como clientes en redes de Windows.
